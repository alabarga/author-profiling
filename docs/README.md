### Features used for author profiling

#### Shortlisted
* Frequency of POS tags
* Capital Letters
* Punctuations
* Number of links
* Readability Score

#### Content based features:
* Bag of words
* Named entities
* Dictionary words
* Slang words
* Contractions
* Sentiment/ Emotion words
* Topics (can be used for LDA)

#### Style based features:
* Frequencies of various POS tags
* Punctuations
